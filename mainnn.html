🎓 EduGrades

Student Report Card Management System

System Features

  • ✅ Add and manage students with class information
  • ✅ Record grades for multiple subjects
  • ✅ Automatically calculate percentages and grades
  • ✅ Generate formal report cards
  • ✅ Print-friendly report card design

To access the application, run:

python app.py

Then open your browser to:
http://127.0.0.1:5001

Quick Start

  • Click "Add Student" to register a new student
  • Click "Grades" to record subject marks
  • Click "Report Card" to view and print results
